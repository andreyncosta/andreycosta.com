# contrato

Client-side rental contract generator at [andreycosta.com/contrato](https://andreycosta.com/contrato).

## What it does

Fills a Brazilian residential rental contract template from form inputs and renders a print-ready legal document — entirely in the browser.

**No server. No storage. No data leaves the client.**

## Fields

- Landlord and tenant personal data (CPF, RG, address)
- Property address and description
- Rental term and monthly value
- Guarantor information (optional)
- Special clauses (optional free text)

## Output

Rendered as a printable HTML page. Use browser print → Save as PDF for a signed copy.

## Legal notice

This generator produces a document for reference purposes. It does not constitute legal advice. Consult a qualified attorney before signing any rental agreement.
